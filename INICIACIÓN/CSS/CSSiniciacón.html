Que es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento estructurado escrito en HTML.

Hay varias formas de utilizar CSS, mas concretamente 3:
1. En el propio documento HTML
2. En un archivo externo con extensión .css
3. (La menos recomendada) Dentro de una etiqueta style en el documento HTML
