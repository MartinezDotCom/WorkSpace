Que es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento estructurado o escrito en HTML.

Hay varias formas de utilizar CSS, mas concretamente 3:
1. En el propio documento HTML
2. En un archivo externo con extensión .css
3. (La menos recomendada) Dentro de una etiqueta style en el documento HTML

La forma que vamos a ver, que es la num. 1 de la lista, es menos mala que la anterior y consiste en
utilizar una etiqueta de STYLE dentro de la etiqueta de HEAD. En este caso no voy a hacer una muestra
debido a que me obligaria a canviar todo el texto ya que al aplicar en la etiquets HEAD afectaria a todo.

Bien, la forma 2 , que es la mas recomendada consiste en importar un archivo de CSS medinte un link
(LINK REL="stylesheet" href="Iniciación.css") que se situa en la etiqueta de HEAD.
Ahora, porque en la etiqueta ponemos esto, el (REL="STYLESHEET") indica que el archivo linked es una
hoja de estilo y el (HREF="Iniciación.css") indica la ruta del archivo de estilo, lo que significa que si
nuestra hoja se llama m... PULPITO.CSS ese es la ruta de archivo que deberos referenciar en HREF.

Siguiendo con la forma 2... hay distintas formas de aplicar dichos estilos, todos se basan
en otorgar propiedades a los elementos de HTML como : ID, CLASS, TAG. A trabes de estas propiedades podemos
modificar los estilos en nuestro documento de CSS, las propiedades siepre deben seleccionarse usando (#).
