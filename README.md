##  Estructura del Repositorio

A continuación se detallan las secciones de este espacio, organizadas por niveles y tecnologías:

### 1. [Iniciación](./Iniciación)
Esta es la carpeta principal de mi etapa pre-estudios. Aquí recopilo **resúmenes prácticos en formato código** basados en los cursos intensivos que realizo de forma autodidacta (YouTube, plataformas online, etc.).

* **Objetivo de esta carpeta:** Crear una base sólida desde cero, estructurar apuntes limpios y servir de guía tanto para mí (como repaso rápido) como para cualquier persona que quiera aprender desarrollo web desde los cimientos.
* **¿Qué encontrarás dentro?**
    * `index.html`: Estructuras base de HTML5, etiquetas semánticas, formularios y buenas prácticas de maquetación.
    * *Próximamente:* Hojas de estilo estructuradas con CSS3 para diseño limpio.
    * *Próximamente:* Lógica y scripts con JavaScript / Python.
